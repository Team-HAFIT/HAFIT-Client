## 프로젝트 개요 👀  

> **해핏**은 Happy와 Fitness의 합성어로, 사용자가 건강하고 즐겁게 운동할 수 있는 환경을 제공하는 AI 포즈 추정 기술(Pose estimation) 기반 `실시간 운동 자세 분석 서비스`입니다.
   
### 🏋️ 주요 서비스

- 불안정한 운동 자세로 인한 `부상을 방지`, 올바른 자세를 유도하기 위한 `시각적•청각적 피드백` 제공
- 반복 횟수 & 세트 수 `자동 카운트`
